# Tourism Intelligence Analytics Platform

基于 Streamlit 的旅游数据分析平台，适合作为统计学与数据分析课程项目。

## 主要功能

- 2017-2025 年 20 个国家的示例旅游数据
- 世界地图与国家数据明细
- 多国家和地区对比
- 同比增长率、相关性与异常年份分析
- 线性回归、移动平均、多项式回归预测
- 排除疫情年份后的回归预测
- 评论分类、好评率与关键词分析
- 自定义旅游数据 CSV 上传
- 自动分析报告生成与下载

## 安装与运行

```bash
pip install -r requirements.txt
streamlit run app.py
```

程序内置课程项目示例数据，因此不上传文件也可以直接运行。评论分析可以上传仓库中的 `reviews_template.csv`。

## 自定义旅游数据

上传文件至少需要以下字段：

- `Country`：国家名称
- `Year`：年份
- `Tourists`：游客人数，单位为人次
- `Revenue`：旅游收入，单位为亿美元

地图功能建议额外提供：`Region`、`ISO3`、`Latitude`、`Longitude`。

## 数据说明

默认数据用于课程项目与功能演示，并非正式官方统计数据。正式研究时请替换为官方统计机构或可靠公开数据库的数据，预测结果仅供趋势参考。
